# 全球星榜潜客 HOT

公开信源驱动的潜在线索看板，包含“精选线索”和“全部动态”两个入口。

## 自动更新

GitHub Actions 每天按北京时间 09:00、12:30、16:00、19:30 自动运行。工作流直接读取公开网页，不需要密钥。

## 当前信源

演出与活动：猫眼演出、大麦、秀动、B站会员购、Bilibili World。

品牌与营销：广告门、SocialBeta、数英、梅花网、美通社亚洲。

娱乐与影视：新浪娱乐、1905电影网。

## 目录

`docs/` 是 GitHub Pages 发布目录。

`scripts/` 保存抓取、去重和精选逻辑。

`config/` 保存信源和信号词配置。

## 数据说明

页面只处理公开网页中已经展示的信息。摘要、机会判断和可信度分数用于线索初筛，正式跟进前仍需回到原始来源核实。
